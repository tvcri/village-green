# Village Green

Village management software for organizing members, volunteers and service requests.

## Project Status

Early development, only handling data synced from Club Express. API and Vue web app scaffolded. Expect breaking changes.

## Quick Start

### Backend
```bash
cd api
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```

## Tech Stack

- **Backend**: Node.js/Express
- **Frontend**: Vue 3 + TypeScript
- **Database**: MySQL
